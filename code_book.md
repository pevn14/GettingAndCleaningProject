# DATA DICTIONARY - 2017 - Human Activity Recognition Using Smartphones Dataset
## Set
#### Identifies the original data set
#### Non Applicable for the narrow data file
#### Value :
* Train
* Set

## Subject
#### Identifies the subject who performed the activity
#### Value :
* range from 1 to 30

## Activity
#### Identify the performed activities
#### Value :
* WALKING
* WALKING_UPSTAIRS
* WALKING_DOWNSTAIRS
* SITTING
* STANDING
* LAYING

## Measurements
#### mean(): Mean value
#### std(): Standard deviation
#### '-XYZ' is used to denote 3-axial signals in the X, Y and Z directions.
#### Features are normalized and bounded within [-1,1]
#### Acc : The acceleration signal from the smartphone accelerometer X axis in standard gravity units 'g'
#### Gyro :The angular velocity vector measured by the gyroscope for each window sample. The units are radians/second.
